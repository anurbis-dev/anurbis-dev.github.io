What / why

Built because pixel art tooling shouldn't need an install.

PixisEditor is a browser-based pixel art editor: layers, frame animation, a dockable tool/panel UI, PNG/GIF export, all running client-side on Canvas 2D. Ships as a single-file build, so it works the same whether it's hosted, downloaded, or opened straight from disk.

It's an independent project developed solo, currently in active beta with new tools and fixes shipping regularly.

renderer Canvas 2D, tiled compositor
stack vanilla ES6, Vite single-file build
runs entirely client-side, no server
status open beta, shipping weekly

What's new

Recent updates

1.13.0 — 2026-07-24

  • Brush stroke Spacing for freehand tools and stamp — smooth lines at low values, dotted stamps when higher.
  • Dither brush overhaul: stroke-locked pattern phase, more geometric presets, live pattern cursor, sticky mirror flip, and hold-to-erase under the pattern mask.
  • Line, rectangle, and ellipse previews match the final stroke pixel-for-pixel (size, shape, symmetry).
  • Process Recorder skips idle pauses so the timeline only counts active drawing time.
  • Canvases that reuse the same sprite stay correct after you edit the source and switch tabs.
  • Layer/frame thumbnails and stamp capture include adjustments and layer effects.
  • Stamp: Esc resets live rotation to 0°.
  • Icon editor: Options tab (mirror and chrome), plus icons for detachable Tool sections.
  • Eyedropper locks the sampled color on click; the brush cursor updates as soon as the color changes.
  • Selection handles: hover scale and click-to-flip; Move tool auto-select, Shift multi-select, and multi-layer drag fixes.

1.12.0 — 2026-07-23

  • Hotkey Editor: full-screen style modal with searchable command list, virtual keyboard, rebind/capture, conflict Take over, export/import, and Empty/Modified/Conflicts filters.
  • Gestures tab in the Hotkey Editor: multi-touch actions with a trackpad diagram, enable/disable, and fixed finger counts.
  • Hold a tool key (e.g. B) to use that tool temporarily, then return to the previous tool on release; quick tap still selects permanently.
  • Temporary hotkeys for brush presets, stamp presets, and palette swatches (right-click → Edit Hotkey…).
  • Icon packs (emoji / mono) plus an icon pack editor to customize tool and panel glyphs.
  • Process Recorder transport: one-row Record / Play / Pause, blink-on-record, pause freezes capture or playback; recorder panel ignored while recording.

1.11.0 — 2026-07-22

  • Process Recorder: record a full drawing session and play it back with a timeline scrubber, speed control, and automatic save when you stop — then export a GIF timelapse of the process.
  • Free panels that sit flush against each other now move as a glued group when you drag one.
  • Toasts can be success/error/warning/info colored, and you can set how long they stay visible.
  • Merge selected layers with a hotkey (Ctrl+E by default), including multi-select collapse.
  • Layer and group notes open in a proper dialog instead of a browser prompt.
  • Temporary eyedropper is a rebindable hold key with a live loupe that follows your pointer.
  • Repaint can lock to the first color under the brush (hold Alt) so only matching pixels recolor.
  • Split view: paint and use the context menu in either pane; better Move-tool pick/flip on sprites.

1.10.0 — 2026-07-22

  • Embed one canvas as a layer inside another, like a reusable sprite — move it, rotate it, flip it, and give it its own independent animation, all while it stays linked back to the original so edits there show up everywhere it's used.
  • The Move tool now also works with regular paint layers, plus drag-to-rotate and quick flip shortcuts, and finally has a default keyboard shortcut.
  • New canvases can go up to 2048×2048 (up from 512×512).
  • Toolbar and panel customization now reliably survives saving/loading projects and switching layouts.
  • Panels can be collapsed with a quick modifier + middle-click, alongside the existing move/resize shortcuts.
