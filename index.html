What / why

Built because pixel art tooling shouldn't need an install.

PixisEditor is a browser-based pixel art editor: layers, frame animation, a dockable tool/panel UI, PNG/GIF export, all running client-side on Canvas 2D. Ships as a single-file build, so it works the same whether it's hosted, downloaded, or opened straight from disk.

It's an independent project developed solo, currently in active beta with new tools and fixes shipping regularly.

renderer Canvas 2D, tiled compositor
stack vanilla ES6, Vite single-file build
runs entirely client-side, no server
status open beta, shipping weekly
Features

Everything a pixel workflow needs

No plugins to hunt down, no export pipeline to wire up.

▤

Layers & groups

Full layer stack with groups, blend modes, opacity and adjustment layers.

▶

Frame animation

Frame-by-frame animation with onion skinning, right in the same canvas.

✎

Full tool set

Brush, shapes, lasso, retouch and palette tools with per-tool settings.

▦

Dockable panels

Rearrange tool, layers and preview panels into the layout you want.

⇩

PNG / GIF export

Export finished art or animations without leaving the browser.

⚡

Single-file, no install

One self-contained build — open it from a URL or straight off disk.
